# タッチ！ことばあそび 🐶🍎🚗

あかちゃん・こども（1〜2歳）向けの知育あそびです。
**がめんのどこをタッチしても**、その場所にひらがなの「ことば」と「え（絵文字）」が
ぽよんと出てきて、**日本語の声で読み上げ**＋ポップ音＋紙吹雪でお祝いします。

- まちがいなし。タッチすれば必ず楽しい反応が返ります（あかちゃん向けの作り）
- どうぶつ🐶・たべもの🍎・のりもの🚗 をミックスで約50ことば
- 画像・音声ファイル不要（絵文字＋音声合成＋効果音をブラウザ内で生成）

---

## いますぐ遊ぶ（Windows / パソコン）

`index.html` をダブルクリックしてブラウザで開くだけです。
（推奨：Microsoft Edge または Google Chrome）

- 画面のどこかをクリック／タッチすると、ことばが出ます
- 「音のオン／オフ」は、右下の 🔊 ボタンを **長押し（約0.7秒）** で切り替えます
  （あかちゃんが連打しても勝手に音が消えないよう、長押し式にしています）

> メモ：ブラウザによっては、音声読み上げに対応していないと「声」が出ないことがあります。
> その場合も、絵・文字・効果音・紙吹雪は楽しめます。

## iPhone / iPad で遊ぶ（将来の本番想定）

このアプリは **PWA（ホーム画面に追加してアプリ風に全画面で遊べる）** 構成です。
iPhone で遊ぶには、ファイルをネット上に置く（HTTPSで配信する）必要があります。おすすめの方法：

### かんたんな配信方法の例
1. **GitHub Pages**（無料）… このフォルダをGitHubに上げて Pages を有効化
2. **Netlify / Cloudflare Pages / Vercel**（無料）… フォルダをドラッグ＆ドロップで公開

### iPhoneでホーム画面に追加
1. Safari で公開したURLを開く
2. 下の「共有」ボタン → **「ホーム画面に追加」**
3. ホーム画面のアイコンから起動すると、**全画面のアプリ**のように遊べます

> iOS では、最初に画面を一度タッチしたときから音・声が鳴り始めます（iOSの仕様）。

## ローカルで「iPhone本番に近い形」で確認したいとき

ファイルをそのまま開く（file://）ではなく、簡易サーバーで配信すると、
PWA（manifest）やiOSの挙動を実機に近い形でテストできます。

Python が入っていれば、このフォルダで次を実行：

```bash
python -m http.server 8000
```

→ 同じWi‑FiのiPhoneの Safari から `http://<パソコンのIPアドレス>:8000/` を開く
（※ホーム画面追加でフル機能にするには本番はHTTPS配信が必要です）

## ファイル構成

| ファイル | 役割 |

## あとから変えたいとき（メモ）

- **ことばを増やす／変える**：`index.html` の `WORDS` の配列を編集
  （`{w:"ひらがな", e:"絵文字"}` の形）
- **声の速さ・高さ**：`speak()` の `u.rate`（速さ）, `u.pitch`（高さ）
- **色**：`COLORS` の配列
